# Backend Tools – Documentación

Este documento explica herramientas comunes utilizadas en el desarrollo de APIs y backend.

---

## Axios

Axios es una librería de JavaScript que permite hacer peticiones HTTP desde el frontend o backend de forma sencilla.

Se usa porque:
Simplifica el uso de fetch
Maneja automáticamente JSON
Permite interceptores de peticiones y respuestas
Facilita el manejo de errores

---

## Postman

Postman es una herramienta para probar APIs.

Se usa porque:
Permite enviar peticiones GET, POST, DELETE, etc.
Facilita el testeo del backend sin frontend
Ayuda a depurar errores en la API
Permite guardar colecciones de pruebas

---

## Sentry

Sentry es una plataforma de monitorización de errores.

Se usa porque:
Detecta errores en tiempo real
Muestra trazas detalladas de fallos
Ayuda a mejorar la estabilidad de la aplicación
Permite alertas automáticas

---

## Swagger

Swagger es una herramienta para documentar APIs REST.

Se usa porque:
Genera documentación automática de endpoints
Permite probar la API desde el navegador
Mejora la comprensión del backend
Facilita el trabajo en equipo
